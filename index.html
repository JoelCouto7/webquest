Actividad 2. Diseño de una Webquest

Introducción | Tarea | Proceso | Resultado | Recursos | Evaluación | Conclusión

INTRODUCCIÓN

En esta WebQuest utilizarás los lenguajes HTML, CSS y JavaScript para crear una página web con un elemento interactivo.

TAREA

La tarea consiste en crear una página web que muestre un cuadrado que cambia de color al pulsar un botón. Para completar la tarea debes acceder a los recursos aportados. La finalidad de la tarea es que veas cómo interactúan los tres lenguajes para conformar una página web.

PROCESO

El proceso para completar la tarea es la siguiente:
  • Crear un elemento en HTML.
  • Dar aspecto a dicho elemento en CSS.
  • Crear un evento con JavaScript.
  • Codificar el cambio que produce el evento en JavaScript.

RESULTADO

Haz click en el recuadro para cambiarlo de color.

EVALUACIÓN

Criterio Suficiente Sobresaliente
Elementos HTML Hay 1 elemento HTML. Hay 2 o más elementos HTML.
Estilos CSS Hay 1 elemento con estilo visible. Hay 2 o más elementos con alguna modificación de estilo visible.
Eventos JavaScript Hay 1 interacción con un elemento. Hay 2 o más interacciones con los elementos.

CONCLUSIÓN
