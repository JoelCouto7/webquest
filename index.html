INTRODUCCIÓN

¿Alguna vez te has preguntado cómo hacer una página web?
¿Sabes crear elementos interactivos?
En esta WebQuest utilizarás los lenguajes HTML, CSS y JavaScript para crear una página web con un elemento interactivo.

TAREA

La tarea consiste en crear una página web que muestre un cuadrado que cambia de color al pulsar un botón.
Puede parecer difícil, pero no lo es tanto ya que posees toda la información que internet puede darte. En otras palabras, para completar la tarea debes acceder a los recursos aportados.
La finalidad de la tarea es que veas cómo interactúan los tres lenguajes para conformar una página web.

PROCESO

El proceso para completar la tarea es la siguiente:
  1. Crear un elemento en HTML.

    2. Puedes utilizar cualquier etiqueta HTML: p, a, div, button, ...
    En los recursos tienes un enlace sobre la etiqueta div.

  2. Dar aspecto a dicho elemento en CSS.

    3. Puedes agregar cualquier estilo: background, border, width, ...
    Los estilos deben ser visibles.
    No es necesario que todas las etiquetas tengan estilo, pero mínimo una.
    En los recursos tienes un enlace sobre los colores en CSS.

  3. Crear un evento con JavaScript.

    4. Define qué clase de evento se active en la etiqueta HTML.
    Puede ser: onclick, onmouseover, ...
    En los recursos tienes un enlace sobre eventos en JavaScript.

  4. Codificar el cambio que produce el evento en JavaScript.

    5. Debes crear la función que se activa con el evento anterior.
    La función debe modificar el estilo de una etiqueta HTML.
    La modificación de estilo debe ser visible.
    En los recursos tienes ejemplos de eventos que modifican el estilo de una etiqueta HTML.


RESULTADO

Haz click en el recuadro para cambiarlo de color.

EVALUACIÓN

Criterio Suficiente Sobresaliente
Elementos HTML Hay 1 elemento HTML. Hay 2 o más elementos HTML.
Estilos CSS Hay 1 elemento con estilo visible. Hay 2 o más elementos con alguna modificación de estilo visible.
Eventos JavaScript Hay 1 interacción con un elemento. Hay 2 o más interacciones con los elementos.
